The odds probability calculator can convert the odds of winning and losing to the probability of winning and losing. Learn the differences between odds and probability.

[https://www.calculator.io/odds-calculator/](https://www.calculator.io/odds-calculator/)

== Usage ==

Add the Odds Calculator shortcode to your page, post or sidebar:

`[ci_odds_calculator]`

Add the following code to your template where you would like the Odds Calculator to appear:

`<?php display_ci_odds_calculator(); ?>`

== Installation ==

1. Upload the Odds Calculator /ci_odds_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Odds Calculator plugin through the "Plugins" menu in WordPress.
