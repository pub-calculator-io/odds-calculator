[https://www.calculator.io/odds-calculator/](https://www.calculator.io/odds-calculator/)

== Usage ==

Add the Odds Calculator shortcode to your page, post or sidebar:

`[calcio_odds_calculator]`

Add the following code to your template where you would like the Odds Calculator to appear:

`<?php display_calcio_odds_calculator(); ?>`

1. Upload the Odds Calculator /calcio_odds_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Odds Calculator plugin through the "Plugins" menu in WordPress.
